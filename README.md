# web-vue
vue初学者搭建简单结构，不需要webpackd，node.js 等脚手架操作

* 目录结构

        ├──web-vue
        │	├── css 
        │	├── images
        │	├── js
        │	│  ├── components.js 组件的方法
        │	│  ├── config.js 全局的一些基本配置
        │	│  ├── index.js 初始化
        │	│  ├── routes.js 所有的路由
        │	│  ├── utils.js 一些常用的方法
        │	├── lib  //公用的核心插件
        │	├── tpl  //存放局部模块的模板
        │	└── index.html 

* 结构非常简单，易上手
